# C-Shell-Emulator
Simple shell emulator in C that mimics basic functionalities similar to a Python shell.
The shell allows users to perform arithmetic operations, handle variables, and provide
help functionality.

# Quick start
1. Make sure you have gcc, make and GNU readline installed

2. Run the following

```
make
cd bin
./py_shell
```

or run a script from file

```
./py_shell <path-to-file>
```

3. For tests run
```
make test
```
